# Template Repository Agent Rules

Single source of truth for all agents working in this repository.

## Read first

Before starting any task:

- `README.md` — overview of this repository and its template sets
- `TEMPLATE_INSTRUCTIONS.md` — tag syntax, configurable values, and adoption process

## Permitted

- Read, create, edit, and delete files in this repository.
- Comment on or request changes to pull requests when instructed by a human.
- Approve a pull request when explicitly instructed by a human.
- Create or switch git branches when explicitly instructed by a human.
- Commit or push changes when explicitly instructed by a human.
- Merge pull requests when explicitly instructed by a human.

## Prohibited

- Proactively offer or ask to commit, push, branch, or open a pull request.
- Modify template documents in `SolidJS/docs/` or `React/docs/` without considering parity: any change that applies to both template sets must be made to both in the same changeset.
- Apply a change to one template set that belongs in both without flagging the omission to the human.

## Escalation

When a situation is not covered by these rules: stop, describe the gap clearly, and await explicit human instruction. Never assume and proceed silently.

## Violations

Surface any constraint violation as a blocking error. Never silently skip or work around a constraint.
